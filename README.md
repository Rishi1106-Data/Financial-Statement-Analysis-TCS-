# 📊 TCS Financial Analysis Project

## 📌 Overview
This project analyzes the financial performance of Tata Consultancy Services (TCS) using real-world financial datasets.

## 🎯 Objectives
- Analyze revenue and profit trends
- Compare profit vs cash flow
- Evaluate financial efficiency and stability

## 🛠 Tools Used
- Python (Pandas, Matplotlib)
- Excel
- SQL (conceptual)

## 📊 Key Insights
- Revenue shows consistent growth
- Profit trend is stable with slight fluctuations
- Cash flow differs from profit, indicating working capital effects
- Expense ratio remains controlled

## 📈 Visualizations
Graphs included:
- Revenue Trend
- Profit Trend
- Profit vs Cash Flow
- Profit Margin
- Expense Ratio

## 🚀 How to Run
```bash
pip install pandas matplotlib
```

Then run the notebook.
