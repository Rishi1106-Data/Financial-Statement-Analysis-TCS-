\# Financial Statement Analysis – TCS Case Study



\## Overview



This project analyzes the financial performance of Tata Consultancy Services (TCS) using real-world financial data. The goal is to evaluate profitability, efficiency, and financial stability through data analytics.



\---



\## Objective



\* Analyze financial performance using key metrics

\* Compare profit vs cash flow trends

\* Identify business insights from financial data

\* Build a structured dataset from raw financial reports



\---



\##  Company



Tata Consultancy Services (TCS)



\---



\##  Tools \& Technologies



\* Python (Pandas, Matplotlib)

\* Excel (Data extraction \& cleaning)

\* SQL (conceptual queries)



\---



\## Dataset



The dataset was created by extracting data from:



\* Profit \& Loss statements

\* Balance Sheets

\* Cash Flow statements

\* Financial ratios



\---



\## Project Workflow



\### 1. Data Collection



\* Extracted data from financial reports (Excel files)



\### 2. Data Cleaning



\* Removed special characters (+, %, commas)

\* Standardized column names



\### 3. Data Integration



\* Merged multiple financial statements into one dataset



\### 4. Feature Engineering



Created key metrics:



\* Profit Margin

\* Expense Ratio

\* Debt-to-Equity

\* Asset Turnover



\### 5. Visualization



\* Revenue trend

\* Profit trend

\* Profit vs Cash Flow

\* Financial ratios



\---



\##  Key Insights



**\* Revenue and profit show consistent growth over time**

**\* Temporary dip in profit suggests external cost pressures**

**\* Profit and cash flow trends differ, indicating working capital impact**

**\* Strong financial stability with controlled expenses**
